# ESP32 Wi-Fi Sensing

Pesquisa sobre Wi-Fi Sensing utilizando sinais CSI (Channel State Information) coletados por dispositivos ESP32 para detecção de presença humana através de modelos leves de aprendizado de máquina.

## Objetivo

Investigar a viabilidade do uso de sinais CSI para detecção de presença em ambientes internos utilizando dispositivos embarcados de baixo custo e técnicas de aprendizado de máquina.

## Temas de Estudo

* Wi-Fi Sensing
* Channel State Information (CSI)
* ESP32
* Processamento de Sinais
* Aprendizado de Máquina
* Detecção de Presença
* Sistemas Embarcados
